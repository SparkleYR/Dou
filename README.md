## Features

- 1Hz refresh - low power consumption
- OpenWeather API integration with auto location detection
- Temperature, humidity, and condition icons
- Album art display from active media
- Elegant play/pause/skip controls with ghost buttons
- Automatic detection of active media sessions
- Silent notification monitoring (no popups during dock mode)
- Grayscale icons for minimal distraction
- Real-time updates without interrupting focus
- Charging state indicator
- Displays upcoming alarm time
- Burn-in prevention with subtle position shifting
- Automatic Do Not Disturb during dock session
- Restores previous DND state on exit

### Setup

1. **Weather API Key**
   - Get a free API key at [OpenWeather](https://openweathermap.org/api)
   - Open Settings (double-tap anywhere in the app)
   - Enter your API key
   - **Note**: New API keys take 10-30 minutes to activate

2. **Grant Permissions**
   - **Notification Access**: Required for notification icons and music controls
   - **Location**: enables accurate weather for your location
   - **Do Not Disturb**: enables focus mode

3. **Usage Tips**
   - Rotate device to landscape for full dock experience
   - double tap anywhere to access settings
   - Keep screen on by connecting to power (or enable in settings)
